# AdManager

Multi-project ad platform for Google Ads and Meta campaigns. Handles all campaign types, keywords, negative keywords, RSAs, and reporting.

## Setup

```bash
# 1. Install dependencies (run from host terminal)
composer install

# 2. Copy and fill in credentials
cp .env.example .env

# 3. Get a refresh token (one-time, run from host terminal)
php bin/auth.php
# Paste the refresh token into .env
```

## Credentials (.env)

```
GOOGLE_ADS_CLIENT_ID=       # OAuth2 Desktop App client ID
GOOGLE_ADS_CLIENT_SECRET=   # OAuth2 Desktop App client secret
GOOGLE_ADS_DEVELOPER_TOKEN= # From ads.google.com/aw/apicenter
GOOGLE_ADS_REFRESH_TOKEN=   # Generated by bin/auth.php
GOOGLE_ADS_CUSTOMER_ID=     # Your Ads account ID (e.g. 374-841-8440)
```

## Commands

| Script | What it does |
|--------|-------------|
| `php bin/auth.php` | One-time OAuth2 flow → generates refresh token |
| `php bin/create-campaign.php` | Interactive campaign builder (Search, PMax, Display, Video, DemandGen) |
| `php bin/add-negatives.php [csv]` | Bulk-add negative keywords from CSV |
| `php bin/report.php [type] [range]` | Performance reports (campaigns/keywords/ads/terms) |

### Report examples

```bash
php bin/report.php campaigns LAST_7_DAYS
php bin/report.php keywords  LAST_30_DAYS
php bin/report.php terms     LAST_7_DAYS    # what people searched
php bin/report.php ads       THIS_MONTH
```

## Source structure

```
src/
  Client.php              # Auth + API client bootstrap
  Keywords.php            # Add keywords + negatives to campaigns/ad groups
  Reports.php             # GAQL reporting (campaigns, keywords, ads, search terms)
  Ads/
    ResponsiveSearch.php  # Create RSAs
  Campaign/
    Search.php            # Search campaigns
    PMax.php              # Performance Max
    Display.php           # Display
    Video.php             # YouTube/Video
    DemandGen.php         # Demand Gen
config/
  campaigns.php           # Auto-generated campaign ID map (gitignored)
```

## Notes

- Campaigns are always created in **PAUSED** state — enable manually after review
- `config/campaigns.php` is auto-created by `create-campaign.php` and gitignored
- Never commit `.env` — it contains live credentials
